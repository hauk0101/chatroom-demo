## 基于 WebSocket 的多人聊天室 demo

#### 项目运行
* 安装 `npm install`
* 运行 `npm start`
* 访问 `http://localhost:3000/`

#### 项目说明

* 技术栈
    * nodejs + express + socket.io
    * layer.js
    * html + css +js

* 目录
 
 ```
  -- public  //客户端静态文件
  -- views   //客户端模板页 
  -- app.js  //express + websocket 服务端主逻辑
 ```
